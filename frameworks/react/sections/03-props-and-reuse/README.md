# 03-props-and-reuse

## 学习目标
- 理解 props 作为配置输入，学会复用 Button、Alert 等通用组件。
- 能解释本节核心概念并在代码中正确落地。
- 能从示例迁移到练习并独立补全 TODO。

## 前置知识
- 已完成前一小节内容。
- 会使用 `pnpm`、`Node.js LTS`、基础 JavaScript（数组/对象/函数）。

## 建议完成顺序
1. 阅读 `basics.md` 建立直觉。
2. 跑通 `example/README.md` 对照结果。
3. 完成 `exercise/README.md` 中 TODO。
4. 对照 `answer/README.md` 复盘差异。

## 预估耗时
- 阅读与理解：20~30 分钟
- 示例动手：20 分钟
- 练习与复盘：30~45 分钟
- 合计：约 1.5 小时

## 本节产物
- `basics.md`：概念与误区说明。
- `example/`：完整可运行项目（含 `src/App.jsx`）。
- `exercise/`：可运行练习项目（`src/App.jsx` 内含 TODO）。
- `answer/`：完整答案项目（可直接运行对照）。

## 验收标准
- 能说清楚“为什么这样设计”。
- `example` 可按说明运行并得到预期行为。
- `exercise` 的 TODO 能独立补全。
- 与 `answer` 对照后能解释差异。
